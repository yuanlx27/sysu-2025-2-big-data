# Data Science and Engineering Practice

This repository contains coursework and lab materials for **Data Science and Engineering Practice** at Sun Yat-Sen University (SYSU), a course focused on the core principles, methods, and engineering techniques of big data.

The course introduces how data science can be applied in complex and diverse data environments, with an emphasis on using practical data science techniques to solve real-world problems.

## Course Overview

The course is organized into two main parts.

### Part I: Data Science Foundations

This part builds the theoretical and conceptual basis for data science.

- Introduction to data science and engineering practice
- Fundamental theories of data science

### Part II: Engineering Practice

This part focuses on applied big data systems and practical data science
workflows across different domains.

- Knowledge graph construction and mining: patent big data practice
- Text detection: internet-scale big data practice
- Multimodal data analysis: medical big data practice
- Recommender systems: human-AI matching practice
- Review and extension of engineering practice

## Learning Goals

By working through this repository, students are expected to:

- Understand the foundational ideas behind data science and big data engineering.
- Learn how to analyze and process heterogeneous data in practical scenarios.
- Apply data science techniques to domain-specific problems.
- Build reproducible workflows for experiments, evaluation, and reporting.

## Source

The course structure is based on the course introduction slide and the mind map from the textbook *Data Science and Engineering Practice*.
